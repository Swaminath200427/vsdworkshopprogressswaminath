# swaminath-vsd ## Day 0 - Tools Installation ### Yosys Installation from source for the latest version. This method ensures you have the most up-to-date features. ```bash $ git clone https://github.com/YosysHQ/yosys.git $ cd yosys $ sudo apt install make $ sudo apt-get install build-essential clang bison flex \ libreadline-dev gawk tcl-dev libffi-dev git \ graphviz xdot pkg-config python3 libboost-system-dev \ libboost-python-dev libboost-filesystem-dev zlib1g-dev $ make $ sudo make install ``` ![Yosys output]() Simple installation using the standard package manager. ```bash $ sudo apt-get install iverilog ``` ![Icarus Verilog installation command line output](iverilog%20installationave Installation using the standard package manager. ```bash $ sudo apt update $ sudo apt install gtkwave ``` ![GTKWave installation command line output](gtk [1](https://cdn.tailwindcss.com)
